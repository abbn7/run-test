# 🚀 Start Here — 3 Commands to Live

```bash
# 1. Install dependencies
bun install        # or: npm install

# 2. Initialize git & push to GitHub
git init -b main
git add .
git commit -m "chore: initial portfolio v2"
git remote add origin https://github.com/<your-username>/<repo>.git
git push -u origin main

# 3. Deploy: open https://vercel.com/new and import your repo.
#    vercel.json is already configured — just click Deploy.
```

That's it. See README.md and DEPLOY.md for details.
